## Local Development

Clone this repo.

Install deps:
```console
yarn install
```

Start the local dev environment:

```console
yarn start
```

## Build

Generate static content into the `build` directory:

```console
yarn build
```

## GitHub Pages Deploy

GIT_USER=<GITHUB_USERNAME> yarn deploy
